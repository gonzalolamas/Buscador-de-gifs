# Gif App

<p>App hecha en React con Hooks para buscar gifs animados en la API de Giphy, y por cada búsqueda se crea un historial de lo que vamos buscando.</p>

## Tecnologías

<ul>
    <li>Html</li>
    <li>Css</li>
    <li>JavaScript</li>
    <li>React</li>
</ul>
